# ⚡ AI 日报

每日 AI 新闻自动抓取 · 翻译 · 语音播报 + AI 产品能力横向对比

👉 **[项目主页](https://yusizou-commits.github.io/ai-news-app)**

## 快速启动

```bash
git clone https://github.com/yusizou-commits/ai-news-app.git
cd ai-news-app
python3 launch.py
```

浏览器自动打开 `http://localhost:7788`

## 功能

- **Tab 1 — 产品排行**：8 类 40+ AI 产品，S/A/B 能力评级，今日有新闻时显示 NEW 角标
- **Tab 2 — 今日新闻**：自动抓取 6 个顶级来源，中英双语展示，一键语音朗读

## 依赖

Python 3（macOS 自带），自动安装：`flask feedparser requests googletrans`
